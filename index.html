Tips for using Seattle Green Space Finder

  • Pressing sort by distance allows for the map to use your devices location. Be sure no pin is active!
  • Drag and drop a pin on the map to change your location to anywhere in the Seattle area.
  • You have the option to set a distance radius using the buffer feature. Even works with a pin!
